## खण्डः ब (Section B)

*(नोट: गद्यांशं पठित्वा प्रश्नानां उत्तरं संस्कृतभाषायां लिखितव्यम्।)*  
**10 अङ्काः**

- (क) अर्जुनः कीदृशः आसीत्?

- (घ) स्वामी कीदृशः आसीत्?  
- (ङ) स्वामी किं चिन्तयति?
